Array

An array is a collection of similar data elements.

These data elements have the same data type. The elements of the array are stored in adjacent memory locations and are referenced by an index (also known as the subscript).

Arrays are generally used when we want to store large amount of similar type of data. But they have the following limitations:

  • Arrays are of fixed size.
  • Data elements are stored in contiguous memory locations (stored one after the other without any gaps) which may not be always available.
  • Insertion and deletion of elements can be problematic because of shifting of elements from their positions.

An array marks of size 10 is declared. The first element will be stored in marks[0], the second element in marks [1], so on and so forth.

Therefore, the nth element is stored in (n - 1)th position of marks

The are fundamental for managing buffers for input/output operations, representing matrices and grids in mathematical and graphical applications and dynamically allocating memory when the size of the data set is not known beforehand.


Linked Lists

A linked list is a very flexible, dynamic data structure in which elements (called nodes) form a sequential list.

In comparison to arrays, a programmer need not worry about how many elements will be stored in the linked list.

Every node in the list points to the next node in the list. Therefore, in a linked list, every node contains the following two types of data:

  • The value of the node or any other data that corresponds to that node.
  • A pointer or link to the next node in the list.

The last node in the list contains a NULL pointer to indicate that it is the end or tail of the list. Since the memory for a node is dynamically allocated when it is added to the list, the total number of nodes that may be added to a list is limited only by the amount of memory available


Linked lists are used where the size of the data set may change frequently such as in dynamic data structures like stack, queues, and hash tables.

Linked lists are also utilized in implementing advanced data structures like graph and trees where nodes need to be interconnected flexibly.

Additionally, they find applications in file systems, memory management algorithms and certain types of caching mechanisms.
